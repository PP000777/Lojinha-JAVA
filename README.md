# Lojinha em Java 🛒

Projeto simples feito para aprender Java.

O programa simula uma lojinha no terminal onde é possível:

- Cadastrar produtos
- Acessar menus
- Fazer compras
- Calcular o total

## Como executar

### Compilar
javac project/*.java

### Executar
java project.mainLojinha

## Objetivo

Praticar conceitos básicos de Java:

- Classes
- Métodos
- Scanner
- Arrays
- Menus
